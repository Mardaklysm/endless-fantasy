# Data Modules

Static gameplay tables live here so future changes can inspect only the relevant domain:

- `items.ts`, `spells.ts`, `gear.ts`, `enemies.ts`, and `playerSkills.ts` hold combat/economy definitions.
- `poiServiceProfiles.ts` holds stock/prices for authored POI shop, inn, church, and magic services.
- `battleTables.ts` holds encounter table IDs.
- Existing world, cloud, object, dungeon tile, and character sprite manifests remain in their specialized files.
